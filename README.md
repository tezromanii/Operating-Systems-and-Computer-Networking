# Operating-Systems-and-Computer-Networking

This coursework project demonstrates practical implementation of operating systems and networking concepts using Linux and Cisco Packet Tracer. It includes a Bash script automating file and directory management, system call analysis using strace, and process creation using fork().

Additionally, a fully functional small office network was designed and configured, including DHCP, DNS, and HTTP servers. Network connectivity, domain resolution, and web access were successfully tested across both wired and wireless devices, validating end-to-end system functionality
